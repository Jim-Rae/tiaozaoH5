
# About

此项目是一个名为跳蚤的二手市场平台web app。

## 技术栈

jquery + jquery.lazyload.js + ajax

## 项目运行

```
git clone https://github.com/Jim-Rae/tiaozaoH5.git  

直接运行 tiaozaoH5/pages/index/index.html
```
